# 信息摘要

躲避神 Run Away from God？
约拿书 (Jonah) 1:1-16

1. 我们为何躲避神？Why do we run away from God? (拿1: 1-2)

2. 躲避神的后果 The results of running away from God
    a.  消极的 Negative (拿1: 4)
    b.  积极的 Positive (拿1: 16)

3. 从“躲避神”到“冲向神” From “running away from God” to “running to God”

结语 Conclusion:
诗篇119: 32 你開廣我心的時候，我就往你命令的道上直奔。I will run the way of thy commandments, when thou shalt enlarge my heart.

線上奉獻: 點這裡, Fund: Chinese Missions Group
	郵寄奉獻: Immanuel Baptist Church.

奉献是基督徒崇拜的行动，表示自愿将从神而来的金钱，一部分归回给神，用在教会圣工上。敬请来宾及慕道朋友不必要勉强奉献。

1. 背诵经文
管家生活：
“各人要照所得的恩赐彼此服事，作神百般恩赐的好管家。”（彼前4:10）
管家生活：
“万军之耶和华说：你们要将当纳的十分之一全然送入仓库，使我家有粮，以此试试我是否为你们敞开天上的窗户，倾福与你们，甚至无处可容。”（玛3:10）
荣神益人：
“你们的光也当这样照在人前，叫他们看见你们的好行为，便将荣耀归给你们在天上的父。”（太5:16）
2. 请准备本课“第一、第二和第三点“。细读这部分的内容，完成每节经文的填空，并将你自己的心得体会写在空白处。
3. 请思考以下问题：
（1）什么叫管家职分？管家职分的意义是什么？
（2）在一天二十四小时中，我用多少时间来做福音事工或行善？
（3）我有哪些恩赐或才能可以奉献于福音事工？该怎样去作？
